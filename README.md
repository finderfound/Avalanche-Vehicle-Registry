# Avalanche Vehicle Registry

Smart contract for peer-to-peer vehicle title transfers on Avalanche. Built for Avalanche Build Games Round 2.

## Deployed Address
Fuji Testnet: [0x257674cC71476003D94F70795Db06feBdFA07C1d](https://testnet.routescan.io/address/0x257674cC71476003D94F70795Db06feBdFA07C1d)

## Features
- ERC721-based digital titles.
- Roles: Manufacturer, Private Owner, etc.
- Private transfers with ERC20 payments.
- VIN/mileage updates.

## Setup
- Compiler: Solidity 0.8.20 (Shanghai EVM).
- Deploy via Remix on Fuji.

## Testing
1. Connect to Fuji (test AVAX from faucet).
2. Grant roles and mint vehicles.
